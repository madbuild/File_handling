 Python File Handling Project

This is a simple file handling project that I built using Python to practice working with files and folders.

The program allows users to perform basic file operations from the command line, such as creating, reading, updating, and deleting files.

## Features

* Create a new file
* Read the contents of a file
* Rename a file
* Overwrite file contents
* Append new data to a file
* Delete a file
* View available files and folders

## Technologies Used

* Python
* pathlib
* os

## What I Learned

While building this project, I practiced:

* File handling in Python
* Functions
* Exception handling using try-except
* Working with the pathlib module
* Reading and writing files
* Basic project organization
